使用hexo和github搭建个人博客

前言

  • 什么是hexo，hexo应该说是静态博客生成器，用户通过写markdown文档的方式发布博文，hexo程序将markdown文档转换成静态页面。
  • 准备工作：
    1. 注册github账号，网站：https://github.com/
    2. 下载node.js，node.js 是一个基于 Chrome V8 引擎的 JavaScirpt 运行环境。网站：http://nodejs.cn/download/

安装完成后输入命令： ssh-keygen -t rsa -C “github上注册的邮箱”然后不断回车。
